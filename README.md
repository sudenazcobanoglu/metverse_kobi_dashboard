# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1OwzcEOdZxO52PO_XhiRtu-Zx0hgNMNY4

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

📊 MetVerse KOBİ Dashboard
Bu proje, Metaverse ekosisteminde faaliyet gösteren KOBİ'lerin dijital performanslarını, satış verilerini ve kullanıcı etkileşimlerini takip edebilmeleri için geliştirilmiş bir Yönetim Paneli (Dashboard) çalışmasıdır.

📌 Önemli Not: Bu çalışma bir ekip projesinin parçası olup; projenin iş analizi, arayüz tasarımı ve dashboard geliştirme süreçleri tamamen tarafımdan yürütülmüştür.

🚀 Projenin Amacı
Yönetim Bilişim Sistemleri perspektifiyle hazırlanan bu dashboard, işletme sahiplerinin karmaşık verileri anlamlandırmasını ve Veri Odaklı Karar Destek mekanizmalarını kullanabilmesini hedefler. KOBİ'lerin dijital dönüşüm süreçlerinde karşılaştıkları veri takibi zorluğunu minimize etmek amacıyla tasarlanmıştır.

🛠️ Kullanılan Teknolojiler
Frontend: HTML5, CSS3, JavaScript

Tasarım: Modern ve kullanıcı dostu UI/UX prensipleri

Versiyon Kontrol: Git & GitHub

💡 Öne Çıkan Özellikler
Veri Görselleştirme: Kritik performans göstergelerinin (KPI) grafik ve tablolarla sunumu.

Kullanıcı Odaklı Arayüz: İşletme sahiplerinin teknik bilgiye ihtiyaç duymadan kullanabileceği sade tasarım.

Modüler Yapı: Farklı KOBİ ihtiyaçlarına göre genişletilebilir kod mimarisi.
